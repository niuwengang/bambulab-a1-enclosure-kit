# 拓竹A1封箱套件
为拓竹A1 3D打印机设计的全包围封箱解决方案

## 特点
- 隔音降噪：有效降低3D打印时产生的噪音
- 空气净化：集成排风系统，排出打印过程中的有害气体
- 美观实用：简洁现代的外观设计
